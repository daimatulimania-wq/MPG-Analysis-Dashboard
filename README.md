# MPG-Analysis-Dashboard

## Deskripsi Proyek
Proyek ini merupakan bagian dari kegiatan **Studi Independen Data Science** di **PT Vinix Seven Aurum**.  
Tujuan proyek adalah menganalisis faktor-faktor yang memengaruhi efisiensi bahan bakar kendaraan (Miles Per Gallon - MPG) menggunakan dataset **Auto MPG** dari UCI Machine Learning Repository. 

Analisis ini bertujuan untuk melihat bagaimana spesifikasi teknis kendaraan seperti berat, tenaga mesin, dan jumlah silinder berdampak pada konsumsi bahan bakar dari tahun ke tahun.

## Tujuan Proyek
- Menganlisis perbedaan rata-rata MPG berdasarkan Cyinders
- Menganalisis hubungan antara Weight dan MPG
- Mengidentifikasi tren rata-rata MPG dari tahun ke tahun

## Metodologi
Proyek ini menggunakan standar **CRISP-DM**:
1. **Business Understanding**: Memahami faktor penentu efisiensi bahan bakar.
2. **Data Understanding**: Eksplorasi dataset Auto MPG (9 atribut, 398 data).
3. **Data Preparation**: Tahap ini bertujuan untuk membersihkan dan mentransformasikan data agar siap digunakan pada tahap analisis eksploratif.`.
4. **Data Analysis**: Melakukan analisis statistik dan korelasi.
5. **Evaluation**: Memastikan visualisasi menjawab permasalahan.
6. **Deployment**: Publikasi dashboard visualisasi interaktif.

## Hasil Singkat (Key Insights)
- Semakin banyak jumlah silinder, rata-rata MPG semakin rendah, namun efisiensi bahan bakar juga dipengaruhi oleh faktor lain seperti berat kendaraan dan horsepower.
- Terdapat hubungan negatif antara berat kendaraan dan MPG. Semakin berat kendaraan, semakin rendah nilai MPG, yang berarti efisiensi bahan bakar menurun.
- Rata-rata MPG menunjukkan tren meningkat dari 1970 hingga 1982, yang menandakan bahwa kendaraan semakin efisien seiring dengan perkembangan teknologi dan regulasi hemat bahan bakar.
